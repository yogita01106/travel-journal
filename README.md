# Travel Journal

A simple and responsive Travel Journal web application built with **React** as part of the *. The application displays a collection of travel destinations with images, locations, Google Maps links, dates, and descriptions using reusable React components and props.

## Features

-  Displays multiple travel destinations
-  Responsive destination cards with images
-  Direct links to locations on Google Maps
-  Built using reusable React components
-  Data-driven rendering using props and arrays
-  Clean and minimal user interface

##  Tech Stack

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## Getting Started

### Clone the repository

```bash
git clone https://github.com/yogita01106/travel-journal.git
```

### Navigate to the project

```bash
cd travel-journal
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
